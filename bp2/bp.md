# 🔥 第 2 页：问题（Problem）

**AI 应用爆发，但 AI 运行方式仍停留在“云服务器时代”**

现实情况：

* SaaS 接 LLM API → 成本不可控
* 数据必须上传云端 → 隐私/合规风险
* 延迟高 → 交互体验差
* 自建 GPU → 重资产、复杂运维

但未来大量 AI 场景发生在：

* Web SaaS
* 浏览器插件
* 企业内部系统
* 边缘设备

**这些场景不适合依赖远程 GPU。**

👉 **AI 已经进入每个应用，但基础运行方式错位。**
